[Elasticsearch] лі„м№ӯ (alias)

  • 2025.07.29
  • ::

#Elasticsearch
[мӣҗліё л§ҒнҒ¬]

лі„м№ӯмқҖ Elasticsearchмқҳ мң мҡ©н•ң ліҙмЎ°кё°лҠҘ мӨ‘ н•ҳлӮҳлӢӨ.
к·ёлғҘ мқҙлҰ„ л¶ҷмқҙлҠ” кІғмқ„ л„ҳм–ҙм„ң, мқјмў…мқҳ лЎңл“ңл°ёлҹ°м„ң л№„мҠ·н•ң м—ӯн• лҸ„ н•ҳкё° л•Ңл¬ёмқҙлӢӨ.

лі„м№ӯмқҙ л¶ҷмқҖ мқёлҚұмҠӨлҠ” лі„м№ӯмқ„ нҶөн•ҙм„ңлҸ„ лҸҷмқјн•ң лҸҷмһ‘мқ„ м·Ён•  мҲҳ мһҲлӢӨ.

л§Ңл“ңлҠ” лІ•мқҖ к°„лӢЁн•ҳлӢӨ.
_aliases APIм—җ мғқм„ұ мҡ”мІӯмқ„ н•ҳл©ҙ лҗңлӢӨ.

мқҙкұҙ market_item_v7_0_0лқјлҠ” мқёлҚұмҠӨм—җ market_itemмқҙлқјлҠ” лі„м№ӯмқ„ л¶ҷм—¬мӨҖ кІғмқҙлӢӨ.


к·ёлҹ¬л©ҙ мқҙл ҮкІҢ alias лӘ©лЎқмңјлЎң мЎ°нҡҢк°Җ лҗҳл©°


aliasлҘј нҶөн•ҙм„ңлҸ„ мқёлҚұмҠӨм—җ лҢҖн•ҙм„ң кІҖмғүмқҙлӮҳ write л“ұмқ„ мІҳлҰ¬н•  мҲҳ мһҲлӢӨ.




лӢӨмҡҙнғҖмһ„ м—ҶлҠ” alias көҗмІҙ

aliasлҠ” м„ңлІ„к°Җ к°ҖлҰ¬нӮӨлҠ” мқёлҚұмҠӨлҘј көҗмІҙн•  л•ҢлҸ„ мҡ©мқҙн•ҳкІҢ мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

elasticsearchлҘј м“°лӢӨліҙл©ҙ н•ҳмң„нҳёнҷҳмқҙ ліҙмһҘлҗҳлҠ” мғҒнғңлЎң мқёлҚұмҠӨлҘј мғҲлЎң л§Ңл“Өл•Ңк°Җ л§ҺлӢӨ. м»¬лҹјмқ„ м¶”к°Җн•ңлӢӨкұ°лӮҳ н•ҙлҸ„ мқёлҚұмҠӨлҘј мғҲлЎң л”°м•ј н•ҳкё° л•Ңл¬ёмқҙлӢӨ.
к·ёлҹ¬л©ҙ л§ӨлІҲ м„ңлІ„к°Җ к°ҖлҰ¬нӮӨлҠ” мқёлҚұмҠӨлҘј л°”кҝ”м„ң л°°нҸ¬лҘј н•ҙм•јн•ҳлҠ”лҚ°, aliasлҘј м“°л©ҙ мқҙкұё к°„лӢЁн•ҳкІҢ н•ҙкІ°н•  мҲҳ мһҲлӢӨ.

alias мІҳлҰ¬ APIк°Җ мӣҗмһҗм„ұмқ„ ліҙмһҘн•ҳкё° л•Ңл¬ёмқҙлӢӨ.
мҳҲлҘј л“Өм–ҙ, market_itemк°Җ к°ҖлҰ¬нӮӨлҠ” мӢӨм ң мқёлҚұмҠӨлҘј market_item_v5.3.0м—җм„ң market_item_v6.0.0лЎң л°”кҫёкі  мӢ¶лӢӨл©ҙ, мқҙл ҮкІҢ н•  мҲҳ мһҲлӢӨ.

{
  "actions": [
    {
      "remove": {
        "index": "market_item_v5.3.0",
        "alias": "market_item"
      }
    },
    {
      "add": {
        "index": "market_item_v6.0.0",
        "alias": "market_item"
      }
    }
  ]
}
