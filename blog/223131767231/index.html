#лҸҷмӢңм„ұ

мұ„л„җмқҖ мқјл°©м Ғмқё лҚ°мқҙн„°мқҳ нқҗлҰ„мқ„ н‘ңнҳ„н•ҳкё°м—җ м Ғн•©н•ң Lock л§Өм»ӨлӢҲмҰҳ мӨ‘ н•ҳлӮҳлӢӨ.
лӮҙл¶Җм ҒмңјлЎңлҠ” MutexлӮҳ м„ёл§ҲнҸ¬м–ҙлҘј мқҙмҡ©н•ҙм„ң кө¬нҳ„лҗҳкіӨ н•ңлӢӨ. к·ёлһҳм„ң мӮ¬мӢӨ Mutexмқҳ кі мҲҳмӨҖ м¶”мғҒнҷ”лқјкі  н•  мҲҳлҸ„ мһҲлӢӨ. (л°ҳл“ңмӢң к·ёлҹ° кІғмқҖ м•„лӢҳ)

мұ„л„җмқҳ мў…лҘҳлҠ” м—¬лҹ¬к°Җм§Җк°Җ мһҲм§Җл§Ң, мқјл°ҳм ҒмңјлЎң к°ҖмһҘ л§Һмқҙ мӮ¬мҡ©лҗҳлҠ” нҳ•нғңлҠ” multi producer single consumer(mpsc)лӢӨ.
м—¬лҹ¬к°ңмқҳ н”„лЎңл“Җм„ңк°Җ лҚ°мқҙн„°лҘј ліҙлӮј мҲҳ мһҲкі , н•ҳлӮҳмқҳ м»ЁмҠҲлЁёк°Җ к·ё лҚ°мқҙн„°лҘј лӢӨ л°ӣмқ„ мҲҳ мһҲлҠ” кІғмқҙлӢӨ.
мқҙкұё к·ёлҰјмңјлЎң к·ёлҰ¬л©ҙ мқҙл ҮлӢӨ.

senderл“ӨмқҖ м–ём ңл“ м§Җ лҚ°мқҙн„°лҘј ліҙлӮј мҲҳ мһҲкі , к·ёкұҙ н•ӯмғҒ мұ„л„җм—җ нҒҗмқҳ нҳ•нғңлЎң м ҖмһҘмқҙ лҗңлӢӨ.
к·ёлҰ¬кі  receiverлҠ” нҒҗм—җм„ң лҚ°мқҙн„°лҘј н•ҳлӮҳм”© кәјлӮҙм„ң мӮ¬мҡ©н•ҳкІҢ лҗҳлҠ” кІғмқҙлӢӨ.

нҒҗм—җ лҚ°мқҙн„°к°Җ м—ҶлӢӨл©ҙ л“Өм–ҙмҳ¬л•Ңк№Ңм§Җ лҢҖкё°н•ңлӢӨ.




мһҘлӢЁм җ

Mutex л“ұмқҳ м—¬нғҖ Lock кө¬нҳ„л“Өкіј л№„көҗн•ҳл©ҙ мӮ¬мҡ©лІ•мқҙ м§ҒкҙҖм Ғмқҙкі  лӢЁмҲңн•ҳлӢӨлҠ” кІғмқҙ мһҘм җмқҙлӢӨ. мқјл°ҳм Ғмқё Lockмқ„ м“ё л•Ң нқ”нһҲ кІӘмқ„ мҲҳ мһҲлҠ” лҚ°л“ңлқҪ л“ұмқҳ л¬ём ңк°Җ нҳ„м ҖнһҲ м ҒлӢӨ.

нҠ№нһҲ Actor Model л“ұмқ„ кө¬нҳ„н•ҳкё°м—җ мҡ©мқҙн•ҳлӢӨ.
read м–‘ лҢҖл№„ write м–‘мқҙ көүмһҘнһҲ л§Һмқ„ кІҪмҡ°м—җлҸ„ мң лҰ¬н•ң нҺёмқҙлӢӨ. лӢ№мһҘ лӢӨ мІҳлҰ¬к°Җ л¶Ҳк°ҖлҠҘн•ҳлҚ”лқјлҸ„ нҒҗмқҳ нҳ•нғңлЎң мқјлӢЁ лӢӨ л°ӣкі  ліҙкё° л•Ңл¬ёмқҙлӢӨ.

н•ҳм§Җл§Ң м „л°ҳм Ғмқё м„ұлҠҘмқҖ Mutexл§Ңмқ„ мӮ¬мҡ©н•ҳлҠ” кІғкіј л№„көҗн•ҳл©ҙ л–Ём–ҙм§Ҳ мҲҳлҸ„ мһҲлӢӨ.




мӮ¬мҡ©лЎҖ (Rust)

Rustм—җм„ң к°„лӢЁн•ң мұ„л„җ кё°л°ҳмқҳ ping-pong н”„лЎңк·ёлһЁмқ„ кө¬нҳ„н•ң мҳҲлӢӨ.

лЁјм Җ м»ЁмҠҲлЁёк°Җ м „лӢ¬л°ӣмқ„ мқҙлІӨнҠёмқҳ нҳ•нғңлҘј м •мқҳн•ңлӢӨ.

к·ёлҰ¬кі  к·ёкұё кё°л°ҳмңјлЎң мұ„л„җ к°қмІҙлҘј мғқм„ұн•ңлӢӨ.

м—¬кё°м—җлҠ” senderмҷҖ receiverк°Җ мһҗлҸҷмңјлЎң мғқм„ұлҗңлӢӨ.
senderлҘј producuerк°Җ лӮҳлҲ м„ң к°Җм§Җл©ҙ лҗҳкі , receiverлҠ” лӢЁмқј consumerл§Ң к°Җм§Ҳ мҲҳ мһҲлӢӨ.

к·ёлҹј мқҙм ң м Җкұё к°Җм ёлӢӨк°Җ лҸҷкё°нҷ”лҘј н•  мҲҳ мһҲлӢӨ.
лӢӨмқҢмқҖ senderк°Җ м „лӢ¬н•ҳлҠ” лӘ…л №мқ„ кё°л°ҳмңјлЎң л¬ёмһҗм—ҙмқ„ м¶ңл Ҙн•ҳкі  лҰ¬мӢңлІ„лҘј мў…лЈҢн•ҳлҠ” лЎңм§Ғмқ„ кө¬нҳ„н•ңлӢӨ.

к·ёлҹј кё°лҢҖн•ңлҢҖлЎң мӢӨн–үлҗ  кІғмқҙлӢӨ.

м „мІҙ мҪ”л“ңлӢӨ.

use tokio::sync::mpsc;

enum Event {
    Message(String),
    Terminate,
}

#[tokio::main]
async fn main() {
    let (sender, mut receiver) = mpsc::channel(100);

    // receiver
    tokio::spawn(async move {
        loop {
            let event = receiver.recv().await.unwrap();
            match event {
                Event::Message(msg) => {
                    println!("message: {}", msg);
                }
                Event::Terminate => {
                    println!("terminate");
                    break;
                }
            }
        }
    });

    // sender
    tokio::spawn(async move {
        sender.send(Event::Message("hello".to_string())).await;
        sender.send(Event::Message("world".to_string())).await;
        sender.send(Event::Terminate).await;
    });

    loop {}
}
