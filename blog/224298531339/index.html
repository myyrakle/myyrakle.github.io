기존에 문서 뷰어로 echo-swagger 모듈을 쓰고있었는데, 업데이트도 안되고 이상한 버그투성이라서 직접 만들었다.
애초에 스웨거 띄우는게 그렇게 대단한 것도 아닌데, 환경이 너무 개판이고 관리도 안되더라.

하는김에 echo v4/v5, gin v1, fiber v2/v3도 일관성 있게 말아봤다.

사용법은 간단하다.
Setup 함수에 기반 경로와 swagger spec 데이터를 집어넣으면 된다. 나머지는 알아서 해준다.

그럼 이렇게 바로 뜬다.


그리고 저 세번째 파라미터에는 yaml을 넣어도 되고 json을 넣어도 된다. 알아서 감지한다.

swaggo를 쓰는 경우에는 json으로 받아서 그냥 던저주면 된다.

최소 버전은 다 최대한 널널하게 잡아놨다.

무지성으로 버전 올려대서 열받은 적이 한두번이 아니었고, 끔찍한 문화라고 생각한다.




Scheme 감지

그리고 사소하긴 한데, 스킴 자동 탐지 기능도 넣어놨다.

이 문서를 https로 접근했으면 https로 켜지고, http로 접근했으면 http로 켜진다.
이거 없으면 은근 불편하다.
