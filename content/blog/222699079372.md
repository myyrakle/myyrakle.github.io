원래 그냥 쓰면 그렇고, CloudFront 등을 이용해서 CDN 캐시 동작을 하도록 하는게 일반적인 사용법이다.

웬만해서 S3는 CloudFront와 붙여쓰는거라고 생각하자.

## S3 설정

우선 캐시를 붙일 버킷에 다음과 같이 버킷 정책을 작성해준다.  

![](/images/img_d7939fc35b22af57.png)

```javascript
{
    "Version": "2012-10-17",
    "Statement": [
        {
            "Sid": "AddPerm",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::버킷명/*"
        }
    ]
}
```
여기서 할건 이게 다다.
