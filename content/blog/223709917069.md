## 설치

다음 사이트에 가서 이메일을 입력하면, 다운로드 링크를 이메일로 보내준다.  
[https://www.anaconda.com/download](https://www.anaconda.com/download)  

![](/images/img_6e3dcdc1d7d8a918.png)

여기서 풀패키지를 원하면 아나콘다.  
최소한만 있어도 되고 빠른 설치와 작은 공간 소모를 원한다면 미니콘다를 사용해야 한다.  

내 경우에는 리눅스라서 인스톨러가 CLI로 실행된다.  
윈도우 같은 경우에는 GUI로 실행된다.

## 프로젝트 시작해보기

프로젝트를 세팅해보자.  

![](/images/img_0253a108949de8a6.png)

![](/images/img_fe711e6c7a1f1c5c.png)
먼저 프로젝트 디렉터리를 적당히 만든다.  
conda는 venv 같은 가상환경을 만들고 관리할 뿐이지, 실제 프로젝트 디렉터리를 만든다거나 하지는 않는다.  

그리고 conda create 명령을 통해서 환경을 초기화한다.  
파이썬 버전 또한 지정해서 생성할 수 있다.  

```javascript
conda create --name conda_start python=3.9
```

![](/images/img_2b925033113fc9e4.png)

그리고 방금 만든 가상 환경을 activate하면, 그때부터 가상환경 세션이 열린다.  

![](/images/img_0848f3a30c48eb6d.png)
이 상태에서의 python 상호작용은 모두 conda 컨텍스트 안에서 이루어지는 것이다.  
파이썬 버전도, 종속성도 모두 conda를 따른다.  

이 상태에서 추가 종속성이 필요하다면, pip가 아니라 conda를 사용하면 된다.  
사용법은 어려울 것이 없다.  

```javascript
conda install 모듈명
```

![](/images/img_976df43a1e221da2.png)

![](/images/img_ea58d2c49a80aac8.png)

그 나머지는 그냥 평소에 파이썬 쓰던대로 쓰면 된다.  

![](/images/img_109adc1cce763589.png)
