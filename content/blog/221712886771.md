외부 ip는 내부 ip와 다르게 직접적인 접근으로 얻어낼 수가 없다.  
라우터가 몇개나 달려있을지, 그 종단까지 다 추적하는 것은 어렵기 때문이다.  

때문에 제대로 된 외부 ip를 획득하려면 외부 서비스를 사용해야 한다.  

![](/images/img_975b7609bad4a6f7.png)
이런거 말이다.  

대충 저런 사이트에서 출력된 ip값을 긁어오면 된다.  

파이썬3에서는 urllib.request 모듈을 사용해서 긁어오면 된다.  
파이썬2는 아마 그냥 urllib에 다른 이름으로 제공될 것이다.  
이런식으로 짜면 된다.  

```javascript
import urllib.request
import json

def get_ip():
    data = urllib.request.urlopen("http://ip.jsontest.com").read().decode('utf-8')
    return json.JSONDecoder().decode(data)['ip']
```

그럼 잘 돈다.
