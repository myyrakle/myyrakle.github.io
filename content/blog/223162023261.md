이전 포스트  
[https://blog.naver.com/sssang97/221718437043](https://blog.naver.com/sssang97/221718437043)  
fuzzing 테스트는 무작위의 값을 기반으로 테스트를 수행하는 것을 말한다.  

cargo-fuzz는 libFuzzer와 llvm 수준 지원을 사용해서 fuzzing 테스트를 구현하는데, 현재 Windows에서는 지원되지 않는 것 같다. 리눅스와 MacOS에서 지원된다.  

<br>

<br>

<br>

## 사전 조건

리눅스나 MacOS여야만 한다. Windows는 WSL를 써도 안된다.  

nightly 버전이 필요하다.  

```javascript
rustup default nightly
```

![](/images/img_5be8afcb84921086.png)

그리고 C++11을 지원하는 컴파일러도 설치되어 있어야 한다.  
보통은 g++이 되겠다.  

```javascript
sudo apt-get install g++
```

그리고 cli 도구를 설치한다.  

```javascript
cargo install cargo-fuzz
```

![](/images/img_2d82127c8f295e30.png)

이렇게 실행이 되면 잘 깔린 것이다.  

![](/images/img_23c8e6ce3411d365.png)
