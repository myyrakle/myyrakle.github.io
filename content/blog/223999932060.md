테이블 구조를 복제뜨는건 간단하다. AS 문을 쓰면 동일한 구성의 테이블이 만들어진다.  

```javascript
CREATE TABLE table_backup AS table
```

복제는 INSERT SELECT 문을 쓰면 된다.  

```javascript
INSERT INTO table_backup 
SELECT *
FROM table
```

![](/images/img_004a5f3c9ab238bf.png)

그럼 이후에는 다시 붓던지, 기존 테이블을 삭제한 이후에 rename을 하든지 하면 된다.  

```javascript
DROP TABLE table; 
RENAME TABLE table_backup TO table;
```
