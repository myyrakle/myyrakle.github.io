## 설치

디펜던시 하나면 된다.  


## 기본 사용법

기본 구조는 대략 아래와 같다.  

```javascript
package main

import (
	"log"

	badger "github.com/dgraph-io/badger/v4"
)

func main() {
	// Open the Badger database located in the /tmp/badger directory.
	// It will be created if it doesn't exist.
	db, err := badger.Open(badger.DefaultOptions("./badger"))
	if err != nil {
		log.Fatal(err)
	}
	defer db.Close()
	// Your code here…
}
```

## 값 쓰기

뱃저db는 모든 읽기/쓰기 작업을 트랜잭션 단위로 실행한다.  
그래서 단순히 값을 하나 깔고싶다 해도 다음과 같이 쓰기 트랜잭션을 열고 써야 한다.  

<br>

```javascript
	// 쓰기 트랜잭션
	if err := db.Update(func(txn *badger.Txn) error {
		// 쓰기 작업 수행
		err := txn.Set([]byte("answer"), []byte("42"))
		return err
	}); err != nil {
		log.Fatal(err)
	}
```

## 값 읽기

값을 읽을때도 마찬가지로 트랜잭션을 열고 읽어야 한다.  

```javascript
	// 읽기 트랜잭션
	if err := db.View(func(txn *badger.Txn) error {
		item, err := txn.Get([]byte("answer"))
		if err != nil {
			log.Fatal(err)
		}

		// 아이템의 값을 가져온다.
		val, err := item.ValueCopy(nil)
		if err != nil {
			log.Fatal(err)
		}

		log.Printf("The answer is: %s\n", val)
		return nil
	}); err != nil {
		log.Fatal(err)
	}
```

## TTL 설정

만료기간을 설정해서 레디스처럼 캐싱용으로 쓸 수도 있다.  
데이터 엔트리에 ttl 옵션을 걸고 쑤셔넣으면 된다.  

```javascript
	// 쓰기 트랜잭션
	if err := db.Update(func(txn *badger.Txn) error {
		// 1시간 동안 유효한 키-값 쌍을 만든다.
		entry := badger.NewEntry([]byte("answer"), []byte("4444")).WithTTL(time.Hour)
		err := txn.SetEntry(entry)
		return err
	}); err != nil {
		log.Fatal(err)
	}
```

기능은 이정도로 최소한만 갖고 있는 편이다.  
막 엄청 복잡하거나 대단한 기능이 있진 않다.
