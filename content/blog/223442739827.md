ArgoCD를 기본설정으로 초기화하면 귀찮게도 SSL이 기본으로 켜져있는 상태다.  

![](/images/img_b4a3882d49471c2f.png)
근데 뭐 내부망이나 로컬에서만 쓸거라면 굳이 SSL을 달아야할 필요도 없고 귀찮기만 한 부분이다.  
저걸 끄는 방법은 그리 어렵진 않다.  

일단 configmap이 이런 식으로 있을 것인데  

![](/images/img_7e91e0ffdea55dba.png)

우리가 필요한건 argocd-cmd-params-cm라는 놈이다.  

```javascript
kubectl get configmap argocd-cmd-params-cm --namespace=argocd -o yaml
```

![](/images/img_263b438613de8d33.png)

가져와서  

```javascript
kubectl get configmap argocd-cmd-params-cm --namespace=argocd -o yaml > configmap.yaml
vi configmap.yaml
```

![](/images/img_77d8b0117e878c08.png)

```javascript
data:
  server.insecure: "true"
```
이렇게 필드 하나 추가하고  

<br>

```javascript
kubectl apply -f configmap.yaml
```
적용하고  

<br>

```javascript
kubectl rollout restart deployment argocd-server -n argocd
```

![](/images/img_b1fbadb29e3af3f5.png)
재부팅해주면 된다.  

<br>

![](/images/img_6009e6611e5ffab3.png)
그럼 이제 그냥 들어가질 것이다.
