air는 reload 기능을 제공하는 보조도구다.  
프로그램을 실행하는 중에 코드를 편집해서 저장하면 자동으로 재실행을 해주는 용도다.  

개발중에 사용하기 좋다.  

<br>

<br>

<br>

## 설치

go install로 바로 설치한다.  

```javascript
go install github.com/cosmtrek/air@v1.49.0
```

![](/images/img_b213d4dcd992e20f.png)

![](/images/img_28f79744b9e7d930.png)
그럼 air라는 명령으로 사용할 수 있게 된다.
