Go에는 enum syntax가 따로 없다.  
C에도 enum은 있는데... 유사-모던-C를 지향하는 Go에서 퇴보한 부분 중 하나다.  

그래서 enum 같은 열거값을 정의해서 쓰려 해도 마땅한 방법이 없다.  
그냥 상수로 정의해서 쓴다.

## const와 iota를 이용한 유사 enum

일반적으로 많이 사용하는 방법이다.  
타입에 기반한 값 범위 제한도 안되고, 자동완성도 불편한 점이 많지만, 고퍼들은 그래도 잘 쓴다.  

그냥 이런식으로 const 변수 그룹을 정의하고, iota 등으로 열거값을 주입하는 것이다.  
별거없다.  

```javascript
const (
	RED   = iota // 0
	GREEN = iota // 1
	BLUE  = iota // 2
)
```

![](/images/img_7baf0187bd75b254.png)

이게 불편하다면 그냥 enum을 위한 구조체를 따로 정의해주거나 해야할 것이다.
