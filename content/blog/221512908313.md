자바에서 스레딩을 담당하는 클래스는 Thread다.  
아래처럼 사용할 수 있다.  

```javascript
import java.util.*;

public class Main
{
    public static void main(String[] args) throws Exception
    {
        System.out.println("## 메인 스레드 시작");

        var threads = new ArrayList<Thread>();
        threads.add(new Thread( ()->System.out.println("첫번째 스레드요") ));
        threads.add(new Thread( ()->System.out.println("두번째 스레드요") ));
        threads.add(new Thread( ()->System.out.println("세번째 스레드요") ));
        threads.add(new Thread( ()->System.out.println("네번째 스레드요") ));

        //전부 시작
        for(var thread : threads)
            thread.start();

        System.out.println("## 메인 스레드 종료");
    }
}
```

그럼 저 스레드들이 다 끝날때까지 메인스레드가 블럭된다.
