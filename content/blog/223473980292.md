```javascript
Warning: Permanently added '...' (ED25519) to the list of known hosts.
```

![](/images/img_3bae7ac0efebd7d1.png)
ssh를 통해 로그인을 할때 이런 오류나 경고 메세지가 뜰 수 있다.  
이건 접속 host는 동일한데 인증 방법이나 패스워드 같은게 바뀌었을때 발생하는 현상이다.  

이 경고를 없애려면 ssh config 파일을 편집해서  

```javascript
sudo vi ~/.ssh/config
```

```javascript
UserKnownHostsFile ~/.ssh/known_hosts
```

![](/images/img_1206a80dbdc3e0d5.png)
이런 행을 추가해주면 된다.
