모듈명은 당연히 json이다.  

파이썬에서 json은 딕셔너리를 통해 구현하면 된다.  

```javascript
import json

# 딕셔너리 형태로 데이터 저장
data = {}
data['name']='john'
data['age']='234'
data['family']=['mother','father', 'gentleman']

# json 형식으로 변환
json_data = json.JSONEncoder().encode(data)

# 출력
print('json: {}'.format(json_data))

<br>

print('\n')

<br>

# 다시 딕셔너리 형태로 복구
origin_data = json.JSONDecoder().decode(json_data)
print('your name: {}'.format(origin_data['name']))
print('your age: {}'.format(origin_data['age']))
print('your family: {}'.format(origin_data['family']))
```

통신도 한번 해보자.  
내가 쓸 일이 있어서 쏘는것만 파이썬으로 하고 받는건 자바로 했다.  
json 쏴주는 서버다.  

```javascript
import json
import socket

ip = ''
port = 12345
my_address = (ip,port)
server_socket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
server_socket.bind(my_address)
server_socket.listen(3)
print('## Server is Running ##')

# 딕셔너리 형태로 데이터 저장
data = {}
data['name']='john'
data['age']='234'
data['family'] = ['mother','father', 'gentleman']

# json 형식으로 변환
json_data = json.JSONEncoder().encode(data)

while True:
    client_socket, client_address = server_socket.accept()
    print('# client connected {}'.format(client_address))
    client_socket.send(json_data.encode())
    client_socket.close()
    print('# closed')
```
