최근에 함수 URL이라는 기능이 추가됐다.  

![](/images/img_888c50d39f1e57ee.png)
문서의 설명에 따르면, API Gateway의 고급 기능이 필요하지 않으면서, 단순한 형태의 접근 URL이 필요할때 적합한 기능이라고 한다.  

API Gateway가 번거롭기도 하고 좀 비싸긴 했다.  
이건 추가로 나가는 비용은 따로 없고, Lambda 호출비용만 나간다는 것 같다.  

생성을 하면, 인증과 미인증을 선택할 수 있다.  
근데 AWS_IAM으로 쓸거면 그냥 aws-sdk로 바로 호출해서 쓰지, 뭐하러 저걸 쓰겠나 싶다.  
거의 NONE으로 사용할 것 같다.  

![](/images/img_cd5bb530f5caaf87.png)

만들면 이런식으로 바로 URL이 뽑히는데,  

![](/images/img_d41b3aba34517e10.png)

바로 잘 들어가진다.  

![](/images/img_cd264adec156240c.png)
끝이다.  

간만에 아주 만족스러운 기능이 생긴 것 같다.
