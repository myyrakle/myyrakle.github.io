Clickhouse는 커스텀 함수를 정의하는 기능을 제공한다.  
PostgreSQL의 pl/pgsql만큼 자유롭고 방대한 기능을 제공하진 않지만 단순한 사용사례에서는 쓸만하다.  

함수를 만드는 방법은 다음과 같다.  

```javascript
CREATE FUNCTION 함수명 AS (파라미터) -> 표현식
```
함수의 반환값을 단일 표현식으로 정의된다.  
if 같은 조건문이나 함수 호출 중첩은 가능하나, 재귀호출이나 루프 처리는 불가능하다.  

그래서 사실 진짜 복잡한 로직을 구현하는건 불가능하고, 숏컷 개념으로 접근하는 편이 좋다.  

다음은 간단한 사용례다.  
URL을 받아서 PATH 부분만 추출하는 함수를 만들었다.
