rust도 C처럼 main이라는 함수로부터 시작이 된다.  

함수는   
fn 함수이름(파라미터) -> 반환타입  
{  
ㅤ함수내용  
}  

으로 구성된다. 반환하지 않을경우 main처럼 반환타입 생략이 가능하다.  

그리고 러스트에서는 함수의 반환 방식이 두가지다.  
다른 언어처럼 return 값; 해도 되고  
세미콜론 생략하고  
값  
해도 반환문이 된다.  

파라미터에 정수를 받을 경우  
fn func(param : i32)  
처럼 쓰면 된다.  
이름이 앞에 오고 타입이 뒤에 온다. i32는 32비트 정수타입이다.  

<br>

let text : &str = "hello world!";  

는 상수 선언 구문이다. 바인딩이라고 표현하기도 한다. 자세한건 나중에.  

<br>

println!("{}",text);  

는 출력구문이다.  
이름 뒤에 !가 오는건 그냥 함수가 아니라 매크로함수다.  
어떻게든 성능을 쥐어짜보겠다는 의지가 드러난다.  
포맷팅은 중괄호로 한다. text 문자열이 {}에 바로 들어가는 것이다.
