fcitx 모듈을 먼저 설치한다.  

```javascript
sudo pacman -S fcitx5-im fcitx-hangul fcitx5-qt fcitx5-gtk
```

그리고 환경설정에서 가상 키보드를 활성화하고  

![](/images/img_8d0633df6b6609a0.png)

입력기 설정을 해준다.  

![](/images/img_e7f549c45b3059ff.png)
한국어가 아니라 한글이어야 한다.  

그러고 나면 Cntrl+Space로 한영 전환이 될 것이다. 원한다면 한영키(오른쪽 Alt키)로 키 전환이 되게끔 설정해도 된다.  

이 상태로도 한글 사용이 되긴 하는데, 가상 키보드가 동작할 때도 있고 아닐 때도 있다.  
항상 동작하게 하려면 다음의 추가 설정이 필요하다.  

환경변수 파일 수정해서 fcitx를 넣어준다.  

```javascript
sudo vi /etc/environment
```

```javascript
GTK_IM_MODULE=fcitx
QT_IM_MODULE=fcitx
QT4_IM_MODULE=fcitx
QT5_IM_MODULE=fcitx
XMODIFIERS=@im=fcitx
```

이것도 수정해주고  

```javascript
sudo vi  ~/.xprofile
```

```javascript
fcitx5 -d
```
재부팅하면 된다.  

그래서 환경변수가 잘 적용되었고  

```javascript
env | grep fcitx
```

![](/images/img_0c4e6fda23c81d7f.png)

아무데서나 한글 입력이 잘 되면 된 것이다.
