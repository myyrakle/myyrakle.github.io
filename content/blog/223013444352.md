어쩌다보니 발생한 오류다.  
모든 쿠버 api 호출에서 오류가 발생한다.  

![](/images/img_2c1bd5c33fa03e55.png)

찾아보니 TLS 인증서 오류라고 한다.  
왜 꼬인건지는 모르겠는데... 이런식으로 쿠버 설정파일을 재생성해주면 해결된다.  

```javascript
mv  $HOME/.kube $HOME/.kube.bak
mkdir $HOME/.kube
sudo cp -i /etc/kubernetes/admin.conf $HOME/.kube/config
sudo chown $(id -u):$(id -g) $HOME/.kube/config
```

그럼 될것이다.  

![](/images/img_5561b6685199df6f.png)
