관련 포스트  
[https://blog.naver.com/sssang97/223865990191](https://blog.naver.com/sssang97/223865990191)  

zram을 쓰려면, 먼저 Linux-커널 수준에서 지원이 되어야 한다.  
해당 모듈이 설치되어있는지 확인하려면 다음 명령을 사용하면 된다.  

```javascript
modinfo zram
```

![](/images/img_55082f09668bb052.png)

zram 활성화는 손으로 세팅할 수도 있는데, 감싸놓은 툴킷을 쓰는 편이 편하다.  
다음 모듈을 설치하고  

```javascript
sudo apt-get install systemd-zram-generator
```

![](/images/img_b6ab92cfb4b7e07b.png)

설정파일을 수정한다.  

```javascript
sudo vim /etc/systemd/zram-generator.conf
```

![](/images/img_5445e6908ccb8e16.png)

```javascript
zram-size = ram * 0.5
compression-algorithm = zstd
```

![](/images/img_a428c17ce973183f.png)
이러면 전체 RAM의 절반만큼을 zram으로 압축한다.  

그리고 데몬을 재시작하게 하면  

```javascript
sudo systemctl daemon-reload
sudo systemctl start systemd-zram-setup@zram0.service

swapon --show
```

![](/images/img_1cae618c9de1ef09.png)
알아서 zram과 스왑 구성을 활성화해준다.  
이러면 잘 된 것이다.
