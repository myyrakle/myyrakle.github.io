Visual Studio(이하 VS)  

C++ 개발환경중에 원탑가도를 달리는 MS의 비주얼 스튜디오다.  
대부분의 학습서들에서 이 툴을 표준으로 사용할 정도로 큰 지분을 차지하고 있다.  

컴파일러는 자체제작한 MSVC를 사용한다.  

장점 : 기능이 풍부함. 편의성 좋음. 한국어 지원. 개인용은 무료.  

단점 : 인텔리센스-타이핑 보조기능이 부실한 편. 무거움.  

<br>

<br>

CLion  

요즘 뜨는 JetBrain에서 개발한 C++ IDE이다.  
자체 콘솔을 탑재하고 있다.  
컴파일러는 gcc도 쓰고 msvc도 쓴다.  

장점 : 인텔리센스 기능이 좋음. 비교적 가벼움.  

단점 : 한글화가 안됨. VS보단 편의성이 떨어짐. 유료임.(학생은 무료사용 가능)  

개인적으로 인텔리센스 기능만 VS에 박아다가 쓸수 있으면 좋겠다.  

<br>

<br>

DevCpp  

가볍게 쓸수있는 오픈소스 툴이다.  
컴파일러는 GCC를 사용한다.  

장점 : 완전무료임. 매우 가벼움. 휴대용이라서 usb에 넣고다니면서 쓸 수 있음. 한글화됨.  

단점 : 버전 업뎃이 된지가 좀 됨.(컴파일러 커스텀은 가능하지만 그럴거면 차라리 다른 에디터를 씀)  
인텔리센스가 많이 부실함.  

<br>

<br>

Code Block  

이것도 오픈소스였던걸로 기억한다.  

디자인이 맘에 안들어서 던졌었는데 나중에 다시 써보고 보강함.  

컴파일러는 Gcc를 사용한다.  

장점 : 완전무료임  

단점 : 디자인 구림. 한글화 안됨.  

<br>

<br>

JEditor  

[http://www.ojang.pe.kr/CALENDAR/](http://www.ojang.pe.kr/CALENDAR/)  

죠우저프라는 분이 만드신 휴대용 IDE.  

컴파일러는 msvc를 사용한다.  

장점 : 무료임. 엄청나게 가벼움(61mb), 한글임.   

단점 : 용량에 비례해 기능이 부실함.
