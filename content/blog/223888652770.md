다른 머신에서 remote 접속을 하게 하려면 옵션을 조금 조정해줘야 한다.  

먼저 users 옵션을 확인한다.  

```javascript
sudo vi /etc/clickhouse-server/users.xml
```

![](/images/img_31b5ce5922bf6e72.png)
사용중인 계정에 이렇게 네트워크 옵션이 있으면 된다. 이건 디폴트로 잘 되어있다.  

그리고 글로벌 설정에 들어가서  

```javascript
sudo vi /etc/clickhouse-server/config.xml
```

![](/images/img_3cc6b47fb4505fd5.png)

![](/images/img_d394ed2459b8e028.png)
listen_host에 달린 주석을 해제한다.  

그리고 서버를 재부팅하면  

```javascript
sudo systemctl restart clickhouse-server
```

될 것이다.
