## 그래프 데이터베이스란?

여기서 말하는 그래프는, 통계에 쓰는 그래프가 아니라 자료구조에서 말하는 그래프다.  

![](/images/img_b14827b34b4d58c9.png)

![](/images/img_1190afb4617d4021.jpg)
이렇게 막 사방으로 엮여있는거 말이다.  

그래서 막 복잡하게 서로 관계성을 갖고 엮여있는 구조를 표현하기 좋다.  

<br>

<br>

<br>

## 언어

그래프 데이터베이스들도 SQL처럼 자기들만의 쿼리 언어를 사용한다.  
넵튠은 그 중에서 Gremlin, openCypher, SPARQL을 지원한다.

## 그렘린 설치하기

gremlin은 특정 그래프 데이터베이스에만 종속되지 않는 범용 그래프 쿼리 언어이자, 도구이기도 하다.  
넵튠도 이걸 지원한다.  

ec2 아마존 리눅스에서 설치하는 법이다.  
아래 커맨드들을 순서대로 잘 따라쳐준다.  

```javascript
sudo yum install java-1.8.0-devel
```

```javascript
sudo /usr/sbin/alternatives --config java
```

```javascript
wget https://archive.apache.org/dist/tinkerpop/3.5.2/apache-tinkerpop-gremlin-console-3.5.2-bin.zip
```

```javascript
unzip apache-tinkerpop-gremlin-console-3.5.2-bin.zip
```

```javascript
cd apache-tinkerpop-gremlin-console-3.5.2
```

```javascript
wget https://www.amazontrust.com/repository/SFSRootCAG2.cer
```

```javascript
mkdir /tmp/certs/
```

```javascript
cp /usr/lib/jvm/jre-1.8.0-openjdk/lib/security/cacerts /tmp/certs/cacerts
```

```javascript
sudo keytool -importcert \
               -alias neptune-tests-ca \
               -keystore /tmp/certs/cacerts \
               -file  /home/ec2-user/apache-tinkerpop-gremlin-console-3.5.2/SFSRootCAG2.cer \
               -noprompt \
               -storepass changeit
```

다 깔았으면 conf 경로에 원격접속용 yaml 구성을 작성한다.  

![](/images/img_0703c82087c580e5.png)

```javascript
hosts: [엔드포인트]
port: 8182
connectionPool: { enableSsl: true,  trustStore: /tmp/certs/cacerts }
serializer: { className: org.apache.tinkerpop.gremlin.driver.ser.GraphBinaryMessageSerializerV1, config: { serializeResultToString: true }}
```
엔드포인트는 아까 넵튠 엔드포인트를 넣어주면 된다.  

그리고 gremlin 쉘명령을 입력해서 콘솔에 들어오고,  

![](/images/img_4690f1259dae083b.png)

다음과 같이 원격접속을 시도해서  

```javascript
:remote connect tinkerpop.server conf/neptune-remote.yaml
```

![](/images/img_c97e75b38bcf67e5.png)
이렇게 뜨면 접속 성공이다.  

원격으로 쿼리를 날릴 수 있게 콘솔모드로 전환하자.  

```javascript
:remote console
```

![](/images/img_e79d8eb1e5ef38de.png)

잘 전환이 됐다면, 다음과 같이 그냥 표현식 계산도 잘 될 것이다.  

![](/images/img_bb14fa4b1036767d.png)
그럼 이제 여기서 그래프 쿼리 식들을 연산하면서 확인해볼 수 있다.
