with은 흔히 말하는 리소스 획득은 초기화(RAII) 패턴을 제공하는 신택스 요소다.  
특정 객체의 생성과 소멸 시점을 제어해서 리소스의 회수를 자동화하는 기능을 사용하는 것이다.  

C#의 using - IDisposable 문과 동등하다.  

예를 들면, 아래와 같이 임시파일을 관리하는 매니저 객체를 만드는 경우를 생각해볼 수 있겠다.  

파일매니저를 사용중일때는 파일스트림을 열어둔채로 사용해야 하고, 사용이 끝난 이후에는 스트림을 닫아서 반환해야 한다. 또 필요에 따라서는 다 쓴 임시파일을 바로 삭제하도록 할 수도 있겠다.  

그래서 with문은 아래와 같은 구조를 따른다.

**    ...**

## with 가능 타입 정의하기

with은 __enter__와 __exit__라는 매직메서드를 통해 정의된다.  
as를 거는 시점에서는 __enter__가 호출돼서 바인딩할 값을 반환해주고, with 종료시 __exit__가 호출돼서 자원을 정리해준다.  

아래는 그에 대한 예시 코드다.  

![](/images/img_a4482c9e585d875f.png)

전체 테스트코드  

```javascript
from random import randint
from io import TextIOWrapper

class TempFileManager:
    filename: str
    file_stream: TextIOWrapper

    def append(self, text: str)->'TempFileManager':
        self.file_stream.write(text)

    def __init__(self): 
        self.filename = str(randint(0, 1000000))      
        pass

    def __enter__(self):
        print("File created: " + self.filename)
        self.file_stream = open(self.filename, "a")
        return self

    def __exit__(self, exc_type, exc_val, exc_tb):
        self.file_stream.close()
        print("File closed: " + self.filename)
        pass

<br>

def main(): 
    with TempFileManager() as file: # with 시작시 __enter__ 호출
        file.append("Hello")
        file.append("World")
        file.append("!")
    # with 종료시 __exit__ 호출

    pass

main()
```
