우선 클라우드플레어로 SSL/TLS->Origin Server 탭으로 이동한다.  

![](/images/img_f6e5b9f4588ad306.png)
그리고 Create Cert.. 버튼을 클릭해서 생성창으로 이동한다.  

<br>

![](/images/img_f039d426452a1700.png)
기본설정으로 두고 발급해도 된다.  
그리고 List the hostname.. 부분에 설정하길 원하는 도메인이나 서브도메인 이름을 넣어준다.  

그러면 인증서 본문값과 프라이빗 키를 줄텐데, 한번 받으면 또 받을 수가 없으니 잘 저장해둔다.  

그리고 AWS Certificate Manager에서 인증서 가져오기로 방금 인증서 정보들을 넣어준다.  

![](/images/img_10dbf5b775cf0356.png)
먼저 본문과 프라이빗을 넣어준다.  
인증서 체인이 좀 귀찮은 부분인데,  

아래 링크로 공용? 인증서 코드를 받아서  
[https://developers.cloudflare.com/ssl/static/origin_ca_rsa_root.pem](https://developers.cloudflare.com/ssl/static/origin_ca_rsa_root.pem)  

```javascript
개인 인증서 텍스트...

공용 인증서 텍스트...
```
의 형태로 이어붙여서 넣어주면 된다.  
개인 인증서가 아까 인증서 발급받을때 받았던 그거다.  

그렇게 만들어주면  

![](/images/img_c6798aed75d51ecb.png)
이렇게 인증서 목록에 잘 뜰 것이다.
