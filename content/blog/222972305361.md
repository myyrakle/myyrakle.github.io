모듈을 올려서 작업을 하다보면, 배포하기 전에 로컬에다 두고 컴파일을 하면서 테스트를 해보고 싶을 수 있다.  
좀 맘에 안들긴 하지만 부분적으로 기능을 제공한다.  

먼저 사용할 모듈을 받아온다.  
어째서인지는 모르겠지만, 일단 원본 종속성이 있어야 로컬 종속성으로 치환이 되는 것 같았다.  

```javascript
go get github.com/myyrakle/go-test-lib@v0.7.0
```

![](/images/img_4337da04569ba65e.png)
그럼 이렇게 잘 뜰 것이다.  

저걸 이제 로컬 경로로 치환해서 사용하도록 하면 된다.  
이런 식으로 go.mod를 수정해주자.  

```javascript
go mod edit -replace 원본=로컬경로
```

![](/images/img_34e19da61c7bb242.png)

![](/images/img_0eb2f82398e9eabc.png)
