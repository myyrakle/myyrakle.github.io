Go는 기본적으로 언어 수준에서 nil(null pointer)를 제공하고, 이를 통해 값의 유무를 판단할 수 있다.  
하지만 nil을 어느정도 대체할 수 있는 기능이 존재하는데, 그게 바로 Zero Value다.  
그냥 "", 0 같은 기본값이 들어있다면 null값으로 간주하는 것이다. 그래서 기본값이 의미가 있는 경우라면 사용하기 곤란하다.  

Go는 명시적으로 값을 할당하지 않아도 대부분 타입에 대한 default 값을 할당해준다.  
정수의 경우에는 0인데, 이걸 reflect의 IsZero 등을 사용하면 generic하게 체크할 수 있다.

그리고 포인터의 경우에는 IsZero 값을 사용하기가 곤란할 수 있는데, 그런 경우에는 포인터를 분기처리해서 사용하면 된다.  
아래와 같이 generic 함수를 만들어 사용하면 쓰기 편할 것이다.  

```javascript
// value 값과 pointer 값에 대해서 모두 Zero 값을 체크함
func IsZero[T any](x T) bool {
	typeInfo := reflect.TypeOf(x)
	valueInfo := reflect.ValueOf(x)
	isPointer := typeInfo.Kind() == reflect.Ptr

	if isPointer {
		if valueInfo.IsNil() {
			return true
		} else {
			return false
		}
	} else {
		return valueInfo.IsZero()
	}
}
```
