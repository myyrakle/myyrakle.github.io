rustlings는 Rust 학습을 위한 단순한 가이드용 프로젝트다.  
터미널 인터페이스를 기반으로 안내를 해준다.  

커리큘럼이 기초부터 쉽게 잘 쓰여있어서 초심자에게는 꽤 적합해보인다.  

<br>

<br>

<br>

## 설치

설치 방법은 간단하다.  

```javascript
cargo install rustlings
```

![](/images/img_30c06bf06cf86f54.png)

![](/images/img_3f84323d900498de.png)
그래서 이렇게 실행되면 잘 깔린 것이다.  

<br>

<br>

<br>

## 프로젝트 초기화

이걸 쓰려면 프로젝트 구성을 해줘야 한다.  
먼저 폴더를 하나 만들고  

![](/images/img_c2192a2fee6aacc0.png)

![](/images/img_37c40703982de564.png)
거기서 init 명령을 때리면  

<br>

![](/images/img_afeb49698535f319.png)
이렇게 뭘 잔뜩 만들어준다.  
exercises의 각 폴더에는 문제 코드들이 들어있고, solutions의 각 대칭 폴더에는 문제의 답이 들어있다.  
컨닝하고 싶으면 보면 된다.  

<br>

<br>

<br>

## 사용해보기

디렉터리로 이동해서 rustlings를 입력하면, 그대로 터미널 기반의 UI가 시작된다.  

![](/images/img_cd7bd3e7e683795f.png)

![](/images/img_212ef5ef568d20ec.png)

![](/images/img_b54afce4d8685ef0.png)
인트로부터 시작해서 하나씩 진행이 된다.  
다음은 n을 치면 되고, 어려우면 h를 쳐서 힌트 요청하면 된다.  

문제 코드들을 보면 이런 식으로  

![](/images/img_4c1b226798ff95dc.png)

![](/images/img_7da73d0dd6e09d61.png)
다 뭔가 컴파일 오류가 난 채로 구멍이 뚫려있는데  

<br>

![](/images/img_d1ac43d3a9ca7b4b.png)
문제가 해결되지 않은 상태에서 next를 누르면 에러 문구가 뜨고, 문제를 해결하고 next를 누르면 다음 문제로 넘어가는 식이다.  

하나씩 순서대로 해보면서 익혀나가면 된다.
