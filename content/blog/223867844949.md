Docker 컨테이너를 계속 빌드해서 오르락 내리락을 하다보면 컨테이너를 비롯한 이미지 데이터가 쓰레기처럼 쌓이게 된다.  

![](/images/img_8d7f106d665d90f0.png)
저기 overlay에 있는게 도커 쓰레기다.  

저걸 지우려면 prune 명령을 쓰면 된다.  
실행 중인 컨테이너에는 영향을 주지 않고, 중지된 컨테이너나 빌드캐시, 버려진 이미지 등을 날려준다.  

```javascript
sudo docker system prune -a --volumes
```
