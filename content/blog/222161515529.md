일단 업데이트부터 치고  

![](/images/img_4791f36a3455368b.png)

아래 도구들을 설치한다.  

```javascript
sudo apt-get install apt-transport-https ca-certificates curl gnupg-agent software-properties-common
```

![](/images/img_31b6d56592b2cbcd.png)

curl로 도커 다운받고  

```javascript
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -
```

![](/images/img_b998675908886207.png)

아래와 같이 설정한 후  

```javascript
sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu focal stable"
sudo apt-get update
```

![](/images/img_d65e96bc0d0c495c.png)

install을 때리면 된다.  

```javascript
sudo apt-get install docker-ce
```

![](/images/img_e5da3160fec785de.png)

그래서 이렇게 잘 실행되면 성공이다.  

![](/images/img_c3ef212cdb97c5b2.png)
