GC의 종류와 내부 구조에 대해서는 별도 포스트를 참조한다.  
[https://blog.naver.com/sssang97/223575258532](https://blog.naver.com/sssang97/223575258532)  
여기서는 그냥 옵션 사용법만 정리한다.  

java는 버전마다 기본 GC 알고리즘이 다르다.  
현재 기본 GC는 다음 명령으로 확인할 수 있다.  

```javascript
java -XX:+PrintCommandLineFlags -version
```

![](/images/img_7e72f33aedc5048e.png)
최신버전이라 g1gc를 쓰고 있다.  

사용 가능한 gc 목록은 이렇게 볼 수 있는데  

```javascript
java -XX:+PrintFlagsFinal -version
```

![](/images/img_442c8c8d782950a4.png)
이건 gc뿐만 아니라 별별 플래그를 다 뿌려주는거라 보기 어렵다.  

적당히 필터 추고 실행하면  

```javascript
java -XX:+PrintFlagsFinal -version | grep -E 'Use.*GC'
```

![](/images/img_003823ac1e8f00b1.png)
이렇게 나온다.  
저기서 GC 옵션을 골라서 실행하면 된다.  

사용법도 어렵지는 않다.  
그냥 java로 바이트코드 실행하는 시점에 옵션 주면 된다.  

```javascript
java -XX:+옵션명 -cp java/basic Main
```

![](/images/img_44d62d60a4f45bec.png)
그렇다.
