serde는 다양한 데이터형식들을 직렬화/역직렬화하게 도와주는 멋진 라이브러리다.  
JSON, BSON, YAML, TOML, URL 등등 대부분의 형식을 지원한다.  

우선은 이중에서 가장 뜨거운 포맷인 JSON을 먼저 다뤄보겠다.  

먼저 serde_json을 종속성에 추가한다.

## 문자열->객체

## json 문자열에서 json 객체를 뽑아내는 것부터 해보자.
json 규격을 충족하는 간단한 형태의 문자열을 만들어봤다.  

![](/images/img_b01d196992090052.png)

저걸 파싱하려면 from_str 등의 함수를 사용하면 된다.  

![](/images/img_0247652ed356376c.png)

Value는 재귀적으로 정의된 열거 타입이다.  
그래서 모든 json 객체는 Value 타입이기도 하다.  

![](/images/img_a1eaaced4b2155b7.png)

여튼, 추출된 Value 객체에서 요소를 가져오려면, 그냥 해당 요소의 이름으로 인덱싱을 하면 된다.  
그럼 다시 해당 요소를 Value로 감싸서 반환하는 것이다.  

![](/images/img_b7dddd3894d80d2e.png)

근데 전부 Value로만 퉁쳐서는 제대로된 처리가 불가능하다.  
Value라는 껍질을 벗겨서 기본타입들로 바꾸려면 대응되는 as_... 메서드들을 사용하면 된다. 이건 Option을 반환한다.  
가령 as_i64는 해당 Value가 숫자일 경우엔 Some(값)을, 아니면 None을 반환하는 것이다.  

![](/images/img_242dbebc4a2a76c0.png)

저대로 돌리면 잘 돌 것이다.  

![](/images/img_be48e87cc57f1fcb.png)

전체 소스는 이렇다.  

```javascript
fn main() 
{
    //json 문자열
    let json = r#"
    {
        "name":"홍길동",
        "age":300,
        "hobby":["영화감상", "등산", "해적질"] 
    }
    "#;

    use serde_json::Value;

    //문자열을 파싱해 json 객체 Value를 생성
    let result:Value = serde_json::from_str(json)
        .expect("파싱 실패");

<br>

    let name:Value = result["name"].clone();
    println!("name:{}", name);

    let age:i64 = result["age"].as_i64()
        .expect("실패");
    println!("age:{}", age);

    println!("hobby:{}", result["hobby"]);
}
```
