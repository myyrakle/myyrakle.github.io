아마존 리눅스에서 도커를 설치하려면, 아래의 명령들을 순서대로 실행해주면 된다.  

```javascript
sudo yum update -y
sudo amazon-linux-extras install docker
sudo yum install docker
sudo service docker start
sudo usermod -a -G docker ec2-user
```

저렇게 해서 오류 없이 docker 명령이 잘 뜬다면 잘 깔린 것이다.
