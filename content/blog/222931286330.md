## Fuzzing 테스트하기

이제 다시 해보자. 기존 테스트용 소스파일에 다음과 같은 형태로 코드를 추가한다.  
함수의 이름은 Fuzz로 시작해야 하고, 파라미터 타입도 F여야 한다.  
그리고 Fuzz 함수를 호출해서, 거기 안에서 진짜 테스트케이스를 작성하면 된다. 2번째 매개변수부터는 대충 추가하면 그 타입에 맞는 값을 알아서 넣어준다.  

![](/images/img_633d875dbc0b2ce9.png)

```javascript
func FuzzTest2(f *testing.F) {
	f.Fuzz(func(t *testing.T, lhs int, rhs int) {
		a := Add(lhs, rhs)
		b := Add(rhs, lhs)

		if a != b {
			t.Errorf("lhs: %d, rhs: %d", lhs, rhs)
		}
	})

}
```
덧셈에 사용할 양변 값을 다 정수로 받아와서 돌리게 했다.  
그리고 교환법칙이 성립하는지를 검증하게 했다. 이건 입력값을 예측할 수 없기 때문에, 이런식으로 좀 우회적 검증을 해야 한다.  

그리고 테스트케이스를 돌릴때 fuzz 옵션을 주기만 하면 된다.  

```javascript
go test -fuzz=Fuzz
```
