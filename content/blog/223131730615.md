Fetch And Add는 Compare And Swap과 비슷한 인스트럭션 수준의 lock-free한 atomic 연산 기법이다.  
그냥도 사용할 수 있고, 스핀락 등을 구현하는데 사용할 수도 있다.  

매커니즘을 C 코드로 나타내면 이렇게 된다.  

```javascript
int fetch_and_add(int* object, int add) {
    int old = *object;
    *object += add;
    return old;
}
```
값을 더하고, 더하기 이전의 값을 반환한다.  

Compare And Swap이 비교-후 수정하는 것으로 실패 가능한 연산이었다면, Fetch And Add는 우선 더하고 보는 좀 무식한 동작을 가진다.  
상황에 맞게 잘 사용하면 된다.
