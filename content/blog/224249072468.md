## 컨셉

superpowers는 그냥 빠르게 찍어내는 것보다는, 체계적인 과정을 통해서 진정으로 완성된 프로그램을 만드는 것을 목표로 한다.  
그래서 다음과 같은 다양한 기술들을 활용한다.  

**브레인스토밍**  
유저와의 반복적인 피드백을 통해 정확한 구현 형태를 구체화한다. 이를 위해서 UI 스켈레톤을 자동으로 만들어주기도 한다.  

**worktree 기반 서브 프로젝트 생성 **  
worktree와 docker 등을 활용해서 별도의 격리된 영역에서 개발 작업을 진행할 수 있다.  

**plan과 task 관리**  
브레인스토밍이 끝나고 요구사항이 완전히 분석되면, plan 과정에서 구체적은 task 단위를 정의한다.  
여기서 개별 task 단위를 서브에이전트로 병렬처리할 수 있다.  

**TDD** (optional)  
빠른 셀프 테스트를 통해 안정적이고 튼튼한 소프트웨어를 만들 수 있도록 한다. 필수는 아니다.

## 설치

claude code를 기준으로 다룬다.  
세션에서 plugin install 명령을 사용하면 즉시 설치할 수 있다.  

```javascript
/plugin install superpowers@claude-plugins-official
```

![](/images/img_71ab9364e3bcf95f.png)

그리고 재시작하면 skillsets을 로드한 채로 실행이 되는 것이다.  

![](/images/img_b23066f955947896.png)
그러면 기본 세팅은 끝이다. 간단하다.  

<br>

![](/images/img_2ca19d1346e4ad3b.png)
그럼 이런 식으로 스킬 커맨드가 잔뜩 추가된다.  
이걸 직접 호출해도 되고, LLM의 판단에 맡겨서 대신 쓰게 할 수도 있다.  

다른 에이전트들도 잘 지원이 된다.  
codex의 경우에는 이런 식으로 설치할 수 있다.  

```javascript
Fetch and follow instructions from https://raw.githubusercontent.com/obra/superpowers/refs/heads/main/.codex/INSTALL.md
```

![](/images/img_7105a596c508ccc9.png)
이후의 사용법은 사실 거의 같다.
