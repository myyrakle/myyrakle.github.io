napi-rs는 Rust 코드를 Node.js에서 사용할 수 있게 해주는 FFI용 라이브러리다.  

보니까 지원 수준도 나쁘지 않다.  

![](/images/img_a11e08356bfd8362.png)

<br>

<br>

## 기본 구성

Cargo.toml에는 이런식으로 디펜던시를 추가해준다.  

```javascript
[lib]
crate-type = ["cdylib"]

[dependencies]
napi = {version="2", features=["async"]}
napi-derive = "2"
tokio = {version="1", features=["fs"]}

[build-dependencies]
napi-build = "1"
```

![](/images/img_323abdbe5e922c4b.png)
tokio나 async features는 필요없으면 빼도 된다.  

그리고 build.rs 스크립트를 루트 경로에 작성한다.  

```javascript
extern crate napi_build;

fn main() {
    napi_build::setup();
}
```

![](/images/img_8dcfd4a2441e891d.png)
이걸 통해서 Node.js와의 연동을 위한 밑작업을 좀 돌리는 구조다.  
저건 바꿀일이 거의 없다.

![](/images/img_440d347daade9c07.png)
