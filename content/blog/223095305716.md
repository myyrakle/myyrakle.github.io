## 배포하기

우선 연결한 github 레포지토리 루트경로에 테라폼 환경이 구성되어있어야 한다.  
내가 사용한 레포지토리는 이랬다.  
[https://github.com/myyrakle/terraform](https://github.com/myyrakle/terraform)  

변수 2개가 있  

```javascript
// 리전
variable "region" {
  description = "region"
  type        = string
}

// 서버명
variable "server_name" {
  description = "server name"
  type        = string
}
```

ec2 하나만 덜렁 띄우는 단순한 코드다.  

```javascript
terraform {
  required_providers {
    # 일종의 라이브러리 로드
    aws = {
      source  = "hashicorp/aws"
      version = "~> 4.16"
    }
  }

  required_version = ">= 1.2.0"
}

provider "aws" {
  region = var.region
}

resource "aws_instance" "server" {
  tags = {
    Name = var.server_name
  }

  ami           = "ami-0e9bfdb247cc8de84"
  instance_type = "t2.micro"
}
```
