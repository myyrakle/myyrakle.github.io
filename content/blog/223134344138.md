Argo Rollouts은 다양한 배포방법에 대한 기능을 제공하는 ArgoCD의 확장 도구다.  
Blue-Green 배포나 Canary, 점진적 배포 등의 배포기법을 간단하게 사용할 수 있게 해준다.  

<br>

<br>

## 설치

클러스터에 들어가서 다음과 같이 설치를 해준다.  

```javascript
kubectl create namespace argo-rollouts
kubectl apply -n argo-rollouts -f https://github.com/argoproj/argo-rollouts/releases/latest/download/install.yaml
```

![](/images/img_26cd203a71656d70.png)

그럼 아래ㅔ와 같이 리소스들이 생성될 것이다.  

![](/images/img_3011c0d50b0ad2cc.png)

그리고 kubectl CLI용 플러그인도 설치해준다.  

```javascript
sudo curl -LO https://github.com/argoproj/argo-rollouts/releases/latest/download/kubectl-argo-rollouts-linux-amd64
sudo chmod +x ./kubectl-argo-rollouts-linux-amd64
sudo mv ./kubectl-argo-rollouts-linux-amd64 /usr/local/bin/kubectl-argo-rollouts

kubectl argo rollouts version
```

![](/images/img_244243ffd3b1d571.png)
이걸 통해서 argo Rollout에 대한 직접적인 제어를 할 수 있다.
