Google LightHouse는 대표적인 웹페이지 품질 검사 도구 중 하나다.  

단순 성능부터 시작해서 검색 엔진 노출 능력이나 스크립트 보안, 일반적인 사용성에 대한 부분을 전반적으로 검사해서 결과를 내려준다.  

Chrome 브라우저에서는 개발자 도구에 내장되어있어 쉽게 사용 가능하다.  

![](/images/img_a7987d76bdf7f65d.png)

사용법은 그다지 어렵지 않다.  
검사하고 싶은 페이지에 가서 "페이지 로드 분석"을 돌리면 된다.  

![](/images/img_7ccd1abeba60fd6b.png)

![](/images/img_04cbe72d90b4b1eb.png)
그럼 약간의 시간이 지난 후에  

보고서를 던져줄 것이다.  

![](/images/img_f70fa81fda972a59.png)
내 사이트의 경우에는 매우 단순한 정적사이트라서 성능 자체는 아주 잘 나왔다.  
사실 API 서버까지 연결하면 현실적으로 100점이 나오기는 힘들다.  

그럼에도 불구하고 이런저런 권장사항들이 나왔는데  

![](/images/img_aba58fd91f5d0610.png)

![](/images/img_27ab17e1d3a6dc9d.png)
이렇게 꽤 꼼꼼하게 개선할만한 부분들을 지적해준다.  

참고하기 좋은 내용들이 많다.
